# Password Generator Java

A professional desktop password generator application built using Java Swing.

## Features

- Generate secure random passwords
- Select password length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include symbols
- Copy password to clipboard

## Technologies Used

- Java
- Java Swing
- Eclipse IDE
